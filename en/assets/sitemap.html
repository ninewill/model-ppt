Accessibility Info

The main layout of this website is based on the principle of universal web accessibility.

  1. Upper menu links area
  2. Middle menu links area
  3. Center main content display area
  4. Bottom menu links area

The shortcut keystroke combinations for this website have been set as follows:
Alt+U (IE), Alt+Shift+U (Firefox)：Upper menu links area: this part includes links to the menu of functions.
Alt+L (IE), Alt+Shift+L (Firefox)：Middle menu links area: this section contains links for each of the menus.
Alt+C (IE), Alt+Shift+C (Firefox)：Center main content display area: this part displays the information content for each web page.
Alt+B (IE), Alt+Shift+B (Firefox)：Bottom menu links area, this part includes links to the menu of functions.
